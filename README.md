# ojt.todoApp2020

2020 年の ojt 用に作成するサンプルアプリです。

## 環境

### アプリケーション

- java 11
- springBoot RELEASE 2.3.0
- gradle
- lombok
- editorconfig
- yarn
- eslint
  - prettier
- webpack

### インフラストラクチャ

- vagrant
- centos/8
- mysql8.0.17

## 使ったサイト

Spring Initializr
https://start.spring.io/
